# 小说转生图模型可识别提示词：全流程标准化方法

核心底层逻辑：**生图模型只能识别「具象的视觉元素」，完全无法理解抽象的文学描写、心理活动、听觉 / 嗅觉叙事、情绪表达**。所有小说内容的转化，本质都是「去文学化→转视觉化→定标准化→锁权重级」的过程，同时兼顾单张图的准确性和系列图的叙事连贯性。

## 一、前置准备：先把小说拆成「生图最小单元」

小说是长文本、多时空、多叙事线的内容，模型无法一次性消化，必须先按「单镜头、单场景、单核心事件、单情绪节点」拆成最小生图单元，避免画面混乱、叙事断层，这也是你之前分段方案的核心优势。

### 拆分标准（严格遵守，避坑核心）

1. **一个单元 = 一个时空**：同一段提示词里，绝对不能同时出现「范阳冬夜军营」和「长安曲江酒肆」两个时空场景，模型会生成元素大杂烩。

2. **一个单元 = 一个核心主体**：要么聚焦安禄山起兵，要么聚焦萧远城楼远眺，要么聚焦长安百姓群像，不要同时塞多个核心人物，导致主体失焦。

3. **一个单元 = 一个镜头节奏**：大场景空镜、人物中景、细节特写必须拆分，不要一段里既要全景要塞，又要手部握剑特写，模型会无法判断画面重心。

4. **拆分时长适配音频**：单个单元的旁白字数控制在 20\-50 字，对应单张图的画面停留时长 8\-15 秒，完全适配有声漫剧 / 小说音频的播放节奏。

## 二、核心转化：把小说文学语言，翻译成模型能懂的「视觉语言」

这是最核心的一步，核心原则是**删掉所有非视觉描写，把抽象内容 1:1 转化为模型可识别的具象视觉元素**，下面是小说高频内容的转化公式，附你原文的对应示例：

|小说原文高频内容|模型无法识别的原因|转化公式 \+ 对应示例|
|---|---|---|
|抽象情绪 / 心理活动<br>（例：从骨髓深处涌上来的寒意、内心的恐惧、宿命的悲凉）|模型看不到「情绪」，只能看到呈现情绪的视觉载体|情绪→光影 \+ 色调 \+ 人物微动作 \+ 环境细节<br>例：「彻骨寒意」转化为「冬夜冷青硬光、人物口鼻呼出的白气、攥紧剑柄发白的指节、暗调沉郁背景、远处烽火的冷红光斑」|
|非视觉感官描写<br>（例：号角声撕裂寒夜、丝竹声声、觥筹交错的喧闹）|模型没有听觉、嗅觉，只能生成对应声音 / 场景的视觉画面|非视觉描写→对应动作 \+ 场景元素<br>例：「号角声撕裂寒夜」转化为「城头士兵吹响青铜号角、北风扯动的猎猎战旗、漫天风雪、军营连绵的火把阵列」|

## 三、标准化提示词结构：按模型权重逻辑排序，避免信息混乱

生图模型的核心规则是**提示词越靠前，权重越高，对画面的影响越大**，必须固定「权重从高到低」的结构，不要乱堆元素。下面是通用万能结构，适配 Midjourney、Stable Diffusion、国内即梦 / 文心一格等所有主流模型，括号内为你的小说示例内容。

### 通用提示词万能模板（按权重排序）

```Plain Text
【画质基础指令】+【固定风格圣经style_bible】+【镜头语言+核心场景】+【核心主体+细节描述】+【光影+色调+氛围】+【画面细节补充】
```

### 每部分的填写规范 \+ 示例

1. **画质基础指令（权重最高，固定不变）**
作用：锁死画面基础质量，避免低画质、模糊、崩坏
通用固定内容：`杰作，最佳画质，8k，超精细细节，极致细节还原，无变形，无穿模`

2. **固定风格圣经（第二权重，系列图必须全程复用）**
作用：保证所有生成图的画风统一，不会出现叙事割裂，你之前的设定完全符合标准
你的内容可直接复用：`晚唐史诗感，工笔\+写实质感，水墨黑、冷钢蓝、暖色灯火三色主调，宽画幅，风雪与火把烟雾质感，室内外冷暖光强对比，画面做旧磨损效果，纸张颗粒质感，复古动画电影美学`

3. **镜头语言 \+ 核心场景（第三权重，定画面框架）**
作用：告诉模型「拍什么景、在哪拍、画面的边界在哪」，是叙事的核心
示例（对应萧远城楼段落）：`冬夜极广中景镜头，平拍视角，唐代长安含光门城楼之上，大景深，前景是城楼城砖，中景是人物，远景是长安皇城灯火与灞桥方向的烽火`

4. **核心主体 \+ 细节描述（第四权重，定画面核心）**
作用：告诉模型「画面里最核心的东西是什么，长什么样」，人物必须复用固定人物卡
示例（对应萧远城楼段落）：`40岁男性萧远，左脸颊一道细疤，高束黑发，身着深靛色唐代兵部侍郎官袍，腰间悬唐制环首刀，背对镜头，侧脸面向画面右侧，双手攥紧腰间佩剑的剑柄，目光望向远方天际`

5. **光影 \+ 色调 \+ 氛围（第五权重，定画面情绪）**
作用：匹配小说的情绪基调，强化叙事感
示例（对应萧远城楼段落）：`冷青月光硬光打亮人物侧脸，下方长安城暖黄灯火漫反射，远处烽火微弱红光映在人物瞳孔，高对比度，低饱和度暗调基底，肃杀沉郁的氛围，画面暗角，戏剧化阴影`

6. **画面细节补充（最低权重，不影响核心框架）**
作用：丰富画面质感，避免画面空洞
示例：`夜风掀动官袍衣摆，人物口鼻呼出细微白气，城砖斑驳风化纹理，远处宫阙飞檐细节`

## 四、配套负面提示词：规避模型生成错误，必加项

负面提示词的作用是**告诉模型「绝对不要生成什么」**，是避免画面崩坏、元素错误、风格跑偏的核心，尤其是系列小说生图，必须固定一套通用负面词，单张图可额外补充专属负面内容。

### 小说古风叙事通用负面提示词（直接复用）

```Plain Text
现代元素，塑料质感，3D建模，卡通，二次元，萌系，美颜，网红脸，变形，畸形，多余肢体，断指，模糊，低画质，马赛克，水印，文字，边框，白边，过曝，死黑，画面杂乱，不符合唐代形制的服饰和建筑，西方面孔，现代建筑
```

### 单张图专属负面词补充示例

- 酒肆群像图：补充`血腥，暴力，兵器，烽火，战争元素`

- 安禄山起兵图：补充`长安建筑，文人，暖光，安宁氛围`

- 萧远特写图：补充`多人，群像，全景，多余背景元素`

## 六、批量小说生图的连贯性保障技巧

1. **固定 3 个核心不变项**：全程复用同一套「风格圣经」、同一套「人物固定卡」、同一套「通用负面提示词」，只更换每一段的「镜头语言、场景、核心动作、光影氛围」，绝对能保证系列图的叙事连贯性。

2. **前后段落加视觉呼应细节**：比如前一段酒肆画面加「窗外天际线的烽火微光」，后一段萧远画面就聚焦「烽火的红光」，让段落之间有视觉锚点，叙事不会断层。

3. **时空转场加标注**：跨时空的段落，在提示词开头加转场标注，比如「画面从风雪范阳渐隐，叠化转场至暮色长安」，适配音频的时间跨度，让画面衔接更丝滑。

4. **同人物镜头递进逻辑**：同一人物的多段画面，严格遵循「全景→中景→近景→特写」的镜头递进，对应人物情绪从平静到紧绷的变化，和小说的心理动线完全匹配。
